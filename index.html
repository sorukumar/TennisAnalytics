About This Project

Tennis Analytics is an open-source hobby project focused on creating professional-quality data visualizations

Data Sources

All visualizations use publicly available tennis data from ATP, WTA, and Grand Slam tournaments, processed and cleaned for analysis. Jeff Sackman's data is used extensively.

Technology

Built with Python, D3.js, JavaScript, HTML, and CSS. All code is open-source and available on GitHub for learning and collaboration.

Contribute

Interested in contributing? Fork the repository, create a pull request, or suggest new visualization ideas through GitHub issues.
